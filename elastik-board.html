    Cloud Sync

    Sync your board across iPhone and Mac in real time using Firebase — free, takes 3 minutes to set up.

    1 Go to console.firebase.google.com → Add project (name it anything, disable Analytics)
    2 In the left menu: Build → Realtime Database → Create database → choose any region → Start in test mode
    3 Copy the URL shown at the top of the database page — looks like https://your-app-default-rtdb.firebaseio.com
    4 Paste it below and tap Connect
